# [arijus.net](https://arijus.net)

my personal blog

## cli

* `node cli list` - list posts
* `node cli add` - add new post

## dev

* `npm i`
* `npm i -g elm`
* `elm package install`
* `npm start` [localhost:8080](http://localhost:8000)
* `npm test`

## prod

* `npm run build`
* `python -m http.server` [localhost:8000](http://localhost:8000)
